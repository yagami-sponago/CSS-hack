了解です！今回はテーマを **「曇りガラスの効果（Frosted Glass Effect）」** にして、CSSで作る練習問題を複数用意します。CSSでは **`backdrop-filter: blur()`** や半透明背景を組み合わせて作ります。

---

### 問題 1：基本の曇りガラス

**問題:**
幅200px、高さ100pxのボックスに、**背景が少しぼやける曇りガラス効果** をつけてください。背景には任意の画像があるとします。

```html
<div class="glass1">曇りガラス1</div>
```

**模範解答:**

```css
.glass1 {
  width: 200px;
  height: 100px;
  background-color: rgba(255,255,255,0.3);
  backdrop-filter: blur(5px);
  border-radius: 10px;
  display: flex;
  align-items: center;
  justify-content: center;
  color: black;
}
```

💡 **コツ:**

* `backdrop-filter: blur()` で背景をぼかす。
* 半透明の背景色 `rgba()` で曇りガラス感を作る。

---

### 問題 2：角丸＋シャドウ付き

**問題:**
曇りガラスボックスに**角丸と軽い影**を追加してください。

```html
<div class="glass2">曇りガラス2</div>
```

**模範解答:**

```css
.glass2 {
  width: 200px;
  height: 100px;
  background-color: rgba(255,255,255,0.3);
  backdrop-filter: blur(8px);
  border-radius: 15px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.2);
  display: flex;
  align-items: center;
  justify-content: center;
  color: black;
}
```

💡 **コツ:**

* シャドウで浮き上がった印象を作るとリアルなガラス感になる。

---

### 問題 3：枠線付きの曇りガラス

**問題:**
ボックスに**白い半透明の枠線**をつけてください。

```html
<div class="glass3">曇りガラス3</div>
```

**模範解答:**

```css
.glass3 {
  width: 200px;
  height: 100px;
  background-color: rgba(255,255,255,0.25);
  backdrop-filter: blur(6px);
  border-radius: 12px;
  border: 1px solid rgba(255,255,255,0.6);
  display: flex;
  align-items: center;
  justify-content: center;
  color: black;
}
```

💡 **コツ:**

* 半透明の枠線でガラス感を強調。
* `rgba()` で背景に馴染む透明度を調整。

---

### 問題 4：ホバーでぼかしを強める

**問題:**
マウスホバー時に、**ぼかしを強く**して曇りガラス感を変化させてください。

```html
<div class="glass4">曇りガラス4</div>
```

**模範解答:**

```css
.glass4 {
  width: 200px;
  height: 100px;
  background-color: rgba(255,255,255,0.3);
  backdrop-filter: blur(5px);
  border-radius: 10px;
  display: flex;
  align-items: center;
  justify-content: center;
  transition: backdrop-filter 0.3s;
}

.glass4:hover {
  backdrop-filter: blur(12px);
}
```

💡 **コツ:**

* `transition` で滑らかに変化させる。
* ホバーでの強弱をつけると立体感が出る。

---

### 問題 5：円形の曇りガラス

**問題:**
円形の曇りガラスボックスを作ってください。直径150px。

```html
<div class="glass5">円形</div>
```

**模範解答:**

```css
.glass5 {
  width: 150px;
  height: 150px;
  background-color: rgba(255,255,255,0.25);
  backdrop-filter: blur(7px);
  border-radius: 50%;
  display: flex;
  align-items: center;
  justify-content: center;
  color: black;
}
```

💡 **コツ:**

* `border-radius: 50%` で円形に。
* 曇りガラス効果は形を問わず適用可能。
