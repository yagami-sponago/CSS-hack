了解です！今回はテーマを **「シンプルな円グラフ」** にして、CSSで作る練習問題を複数用意します。CSSだけで作る場合は **`conic-gradient`** を使うのが基本です。

---

### 問題 1：基本の2色円グラフ

**問題:**
50% 赤、50% 青のシンプルな円グラフを作ってください。サイズは直径150px。

```html
<div class="pie1"></div>
```

**模範解答:**

```css
.pie1 {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background: conic-gradient(red 0% 50%, blue 50% 100%);
}
```

💡 **コツ:**

* `conic-gradient(color1 start% end%, color2 start% end%)` で割合を指定。
* `border-radius: 50%` で円形にする。

---

### 問題 2：3色円グラフ

**問題:**
40% 赤、30% 青、30% 緑の円グラフを作ってください。

```html
<div class="pie2"></div>
```

**模範解答:**

```css
.pie2 {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background: conic-gradient(
    red 0% 40%, 
    blue 40% 70%, 
    green 70% 100%
  );
}
```

💡 **コツ:**

* 各色の終了位置 = 前色の終了位置 + 割合で計算。

---

### 問題 3：白背景＋円グラフ

**問題:**
50% 赤、50% 青の円グラフで、背景を薄いグレーにしてください。

```html
<div class="pie3"></div>
```

**模範解答:**

```css
.pie3 {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background: lightgray; /* 背景 */
  background: conic-gradient(red 0% 50%, blue 50% 100%);
}
```

💡 **コツ:**

* `background` を2重に書くと背景色と円グラフ色を組み合わせられる。

---

### 問題 4：半透明の円グラフ

**問題:**
50% 赤、50% 青の円グラフで、色を半透明にしてください。

```html
<div class="pie4"></div>
```

**模範解答:**

```css
.pie4 {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background: conic-gradient(rgba(255,0,0,0.5) 0% 50%, rgba(0,0,255,0.5) 50% 100%);
}
```

💡 **コツ:**

* `rgba()` を使うと半透明の円グラフが作れる。
* 背景を重ねると下の要素が透けて見える。

---

### 問題 5：中央に穴のあるドーナツグラフ

**問題:**
50% 赤、50% 青の円グラフで、中央に穴を開けたドーナツ型にしてください。

```html
<div class="pie5"></div>
```

**模範解答:**

```css
.pie5 {
  width: 150px;
  height: 150px;
  border-radius: 50%;
  background: conic-gradient(red 0% 50%, blue 50% 100%);
  position: relative;
}

.pie5::before {
  content: '';
  position: absolute;
  top: 25px;
  left: 25px;
  width: 100px;
  height: 100px;
  background-color: white;
  border-radius: 50%;
}
```

💡 **コツ:**

* 擬似要素で中央を白い円で覆うとドーナツ型になる。
* `top/left` と `width/height` の計算で穴の大きさを調整。
